# Sistema de Lista de Tarefas

Sistema web desenvolvido em Python com Flask para cadastro e gerenciamento de tarefas.

## Funcionalidades
- Cadastro de tarefas
- Edição e exclusão
- Reordenação (subir/descer)
- Destaque de tarefas com custo ≥ R$ 1.000,00
- Exibição do total de custos
- Persistência em banco de dados SQLite

## Tecnologias utilizadas
- Python
- Flask
- SQLite
- HTML
